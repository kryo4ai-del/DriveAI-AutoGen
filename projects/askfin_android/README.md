# AskFin Android

Kotlin/Jetpack Compose port of AskFin Premium — AI-powered coaching app for German driver's license exam.

## Tech Stack
- Language: Kotlin
- UI: Jetpack Compose + Material 3
- DI: Hilt
- Async: Kotlin Coroutines + Flow
- Persistence: Room + DataStore
- Navigation: Navigation Compose
- Build: Gradle (Kotlin DSL)
