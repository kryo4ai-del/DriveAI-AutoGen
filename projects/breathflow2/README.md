# BreathFlow2

Eine minimalistische Atem-Uebungs-App. Der Nutzer waehlt eine Uebung (4-7-8 Technik, Box Breathing, oder Einfaches Beruhigen), sieht eine animierte Kreis-Animation die sich mit dem Atem ausdehnt und zusammenzieht, und bekommt einen Timer. Nach der Uebung sieht er wie viele Minuten er diese Woche schon geuebt hat. Alles offline, kein Account, kein Backend.

Built by DriveAI Swarm Factory.
