# GrowMeldAI

Eine intelligente App, die Pflanzen per Kamera erkennt und als persönlicher "Pflanzendoktor" fungiert – mit individuellen Pflegeplänen, Erinnerungen und Problemerkennung.
Hauptfunktionen
1. Pflanzen-Erkennung (AI-Powered)
Kamera-Scan: Foto machen → Pflanze identifizieren
Datenbank: 10.000+ Pflanzen (Zimmerpflanzen, Kräuter, Sukkulenten, etc.)
Details: Name, Herkunft, Schwierigkeitsgrad, Giftigkeit
2. Smarte Pflege-Erinnerungen
Gieß-Erinnerungen (angepasst an Pflanze + Jahreszeit)
Dünge-Zeitplan
Umtopf-Erinnerung
Besprühen/Luftfeuchtigkeit
Standort-Optimierung (Lichtbedarf)
3. Krankheits-Erkennung
Problem-Scan: Gelbe Blätter? Braune Spitzen?
DIagnose: KI erkennt Krankheiten/Schädlinge
Lösung: Konkrete Behandlungsvorschläge
4. Pflanzen-Profil
Eigener "Garten" in der App
Wachstums-Tracking (Fotos über Zeit)
Pflege-Historie
Notizen pro Pflanze
Technische Umsetzung
AI/ML:
Bilderkennung (TensorFlow/PyTorch)
Pflanzen-Datenbank (Plant.id API oder eigene ML-Modelle)
Krankheits-Erkennung (Computer Vision)
Features:
Camera Integration (iOS/Android)
Push Notifications (Erinnerungen)
Offline-Modus (Basis-Daten lokal)
Wetter-API-Integration (angepasste Gieß-Empfehlungen)
