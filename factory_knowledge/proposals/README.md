# Knowledge Proposals

Auto-generated candidate entries from pipeline runs.

These proposals are NOT part of the main knowledge base.
They must be manually reviewed and promoted to `knowledge.json` to take effect.

## File format

Each run that generates proposals creates one file:
`proposal_<run_id>.json`

## Review workflow

1. Read the proposal file
2. Evaluate each candidate entry
3. If valuable: add to `knowledge.json` with appropriate confidence level
4. Delete or archive the proposal file
