- **Daten:**

## Trend 2: Minimalismus als App-Design-Strategie (Anti-Complexity-Trend)

- **Status:** 🟢 Wachsend (strukturell)
- **Daten:**
  - Im Mobile-Gaming-Bereich zeigt sich 2025/2026 deutlich: Downloads sinken (−7% YoY auf 50 Mrd.), aber **Retention und Engagement** steigen als primäre Erfolgskennzahlen. Sensor Tower nennt explizit "live operations, timed events" als Treiber — gleichzeitig wächst der Gegenpol: minimalistische Apps mit klarer Nutzungsroutine. (Quelle: Sensor Tower State of Gaming 2026, games.gg, 01.03.2026)
  - Durchschnittlicher IAP-Umsatz pro Download 2025: **$1.62** — Markt selektiert auf wenige, hochwertige Sessions statt Masse. Das begünstigt strukturell fokussierte Apps mit klarem Nutzenversprechen. (Quelle: Sensor Tower via games.gg, 01.03.2026)
  - ⚠️ Direkter Beleg "Minimalismus als Trend" in Health-Apps: **Nicht in dieser Recherche enthalten.**
- **Quellen:**
  - games.gg: "Mobile Games Reach Record Downloads in 2025" (01.03.2026)
  - Sensor Tower: State of Gaming 2026

## Trend 3: Rückgang von Downloads bei gleichzeitigem Umsatzwachstum — Implikation für Nischen-Apps

- **Status:** 🟡 Ambivalent (strukturell relevant, direkte Übertragbarkeit unklar)
- **Daten:**
  - Mobile Game Downloads: 50 Mrd. in 2025 (−7% YoY seit 2021 kontinuierlich). Google Play: 81% aller Downloads. App Store: 15% Downloads, aber höherer Umsatzanteil und "higher-value users." (Quelle: Sensor Tower via games.gg, 01.03.2026)
  - IAP-Umsatz Mobile Games: **$82 Mrd.** (2025, +1,4% gegenüber 2024). (Quelle: Sensor Tower via games.gg, 01.03.2026)
  - Revenue-Wachstum kommt aus weniger, aber wertvolleren Nutzern — "Monetization First"-Paradigma dominiert 2026. (Quelle: Deconstructorfun.com: State of Mobile 2026, 02.02.2026)
  - ⚠️ Diese Zahlen beziehen sich auf **Games**, nicht auf Health/Wellness-Apps. Übertragbarkeit auf Breathing-App: **strukturell plausibel, aber nicht direkt belegt.**
  - Puzzle-Games IAP-Umsatz: +14% YoY (2025). Strategy-Games IAP: +20% YoY. (Quelle: byyd.me via Sensor Tower, 2026)
- **Quellen:**
  - Sensor Tower: State of Gaming 2026 via byyd.me (03.2026)
  - pocketgamer.biz: "Over 95,000 mobile games were downloaded every minute in 2025"
  - deconstructoroffun.com: "State of Mobile 2026: 9 Trends That Matter" (02.02.2026)
