# Zielgruppen-Profil: GrowMeldAI

Eine intelligente App, die Pflanzen per Kamera erkennt und als persönlicher "Pflanzendoktor" fungiert

---

## Primäre Zielgruppe

| Merkmal | Ausprägung |
|---|---|
| **Alter** | 25–45 Jahre (Kern: Millennials 1981–1994 + junge Gen X 1965–1980) |
| **Geschlecht** | Weiblich dominant (~60–65%) — Pflanzenpflege & Home-Gardening stark weiblich geprägt ⚠️ *Proxy: Gardening-App-Marktdaten + allg. Lifestyle-App-Demografie* |
| **Region(en)** | DACH (Primär), Western Europe, USA/Kanada (Englisch-Rollout Phase 2) — Zimmerpflanzen-Trend besonders stark in urbanen Märkten |
| **Lifestyle-Typ** | Urban/Suburban, Mieter mit Balkonen oder kleinen Wohnungen, Home-Office-affin, Nachhaltigkeit/Natur-bewusst |
| **Spielertyp (App-Analogie)** | **"Casual Caretaker"** — kein Hardcore-Nutzer, aber emotional involviert; ähnlich wie "Casual Candace" (Puzzle/Simulation) aus Mistplay-Report 2024 |
| **Technologie-Affinität** | Mittel bis hoch — nutzt bereits Lifestyle-/Health-Apps (Fitness, Ernährung, Meditation) |
| **Sekundäre Zielgruppe** | 55–70 Jahre (Rentner/Hobbyisten mit Garten) — geringere Tech-Affinität, aber hohe Zahlungsbereitschaft für echten Mehrwert ⚠️ *Proxy* |

## Ausgabeverhalten

| Merkmal | Ausprägung |
|---|---|
| **Durchschnittliche Ausgaben (Non-Game Apps)** | **€3–8/Monat** (Subscription-Modell) — Non-Game Lifestyle-App-Durchschnitt 2024/2025 ⚠️ *Proxy: Durchlauf #004 skillsense Learning + App-Marktdaten* |
| **Ausgabenbereitschaft** | Moderat — Nutzer zahlen für **klaren Nutzwert** (weniger Impulskauf, mehr Rational-Value-driven) |
| **Konversionsrate Free→Paid** | ~3–7% (Lifestyle-/Utility-App-Benchmark 2024) ⚠️ *Branchenproxy* |
| **Bevorzugte Zahlungsmodelle** | 1. **Freemium + Jahres-Abo** (höchste Konversion laut Non-Game-Sub-Trend 2025) 2. **Monatliches Abo** (~€4,99–€9,99) 3. Einmalkauf einzelner Premium-Features (z.B. erweiterte Krankheitserkennung) |
| **Ausgaben-Trigger** | Personalisierte Angebote (+40% Conversion-Lift, Mistplay 2024) + Loyalty/Streak-Mechaniken (51% würden mehr zahlen bei Belohnungssystem) |
| **Spending-Risiko 2025** | ⚠️ 32% aller App-Spender planen Ausgabenreduktion (Mistplay Spender Report 2024) → Jahresabo mit Rabatt als Hedge empfohlen |

## Session-Verhalten

| Merkmal | Ausprägung |
|---|---|
| **Durchschnittliche Session-Länge** | **3–6 Minuten** (Utility-/Lifestyle-App-Proxy; Mobile Gaming: Ø 4–5 Min.) — kein Entertainment-Loop, sondern Task-driven |
| **Sessions pro Tag** | **1–2 Sessions** (primär morgens: Pflanzenkontrolle + abends: Erinnerungen checken) ⚠️ *Proxy: Mobile Utility App Benchmarks* |
| **Sessions pro Woche** | **4–7×** — abhängig von Push-Notification-Frequenz und aktivem Pflanzenpflegebedarf |
| **Peak-Nutzungszeiten** | Morgens 7–9 Uhr (Routine-Check) + Wochenende (Scan neuer Pflanzen, Gartenarbeit) |
| **Nutzungskontext** | Zuhause (Wohnzimmer, Balkon, Küche) — kaum Unterwegs-Nutzung außer beim Pflanzenkauf im Gartencenter |
| **Retention-Treiber** | Push-Notifications (Gieß-Erinnerungen) sind **stärkster organischer Retention-Anker** — ohne sie Drop-off nach Tag 7 wahrscheinlich |

## Social-Verhalten

| Merkmal | Ausprägung |
|---|---|
| **Genutzte Community-Features** | Mittel-niedrig — primär **passive Consumption** (Pflegetipps lesen, Diagnosen teilen) |
| **Sharing-Verhalten** | Instagram/Pinterest: Wachstums-Fotos teilen (hohe organische Virality möglich durch "Vorher/Nachher"-Mechanik) |
| **Referral-Potenzial** | Hoch — 73% der App-Spender installieren Apps via Referral (Mistplay 2024); Pflanzencommunity ist stark word-of-mouth-getrieben |
| **Community-Wunsch** | **"Mein Garten"-Galerie** zum Teilen + Diagnose-Forum ("Kennt jemand diese Krankheit?") |
| **Social-Spending-Einfluss** | Gering für direkte Kaufentscheidungen (69% laut Mistplay: Social beeinflusst Spending kaum) — aber stark für **Discovery/Install** |
| **Influencer-Affinität** | Hoch — Plant-TikTok (#planttok: 10Mrd+ Views) und Plant-Instagram sind organische Marketing-Kanäle |
