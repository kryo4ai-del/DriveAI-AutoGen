# Shared Logs

Dieser Ordner dient zum Austausch zwischen Claude Code Instanzen (Mac + Windows).

- **build_errors.md** – Letzte Build-Fehler
- **status.md** – Aktueller Projektstatus
- **notes.md** – Notizen zwischen den Agents

Beide Claude Code Instanzen können hier lesen und schreiben.
Nach `git push` (Mac) und `git pull` (Windows) ist alles synchron.
