# DriveAI Product Roadmap

---

# Vision

DriveAI will become an intelligent learning assistant for driver's license theory.

Core capability:

Analyze questions from screenshots or photos and explain the correct answer.

---

# Version 1 – MVP (Current Phase)

Goal:

Working app that can analyze theory questions.

Features:

Image Import / Screenshot Analysis  
OCRRecognitionService  
QuestionParsingEngine  
MultipleChoiceDetection  
LLMQuestionSolverService  
Result Screen  
Explanation Output  
Debug Panel  
Demo Quiz Flow  

User Flow:

Dashboard
↓
Import Image
↓
Analysis
↓
Answer + Explanation

Target:

Functional prototype for real testing.

---
